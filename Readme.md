### Cómo utilizar la API localmente

## TENGO QUE PONER ESTO Para el uso de LA aplicaciÓn, a nivel de usuario como de administrador

En la consola, en la ubicación del proyecto (toby-front-2026-1/toby), correr el siguiente comando:

- nvm use 20 (para usar la versión 20)
- yarn dev (para levantar el servidor)

Luego, en el navegador usar la siguiente url:

- http://localhost:5173/

### Correr las pruebas unitarias con Jest

En la consola, en la ubicación del proyecto (toby-front-2026-1/toby), correr los siguientes comandos:

- nvm use 20 (para asegurarnos de estar usando la versión 20)
- npm test

### Usar Linter

En la consola, en la ubicación del proyecto (toby-front-2026-1/toby), correr los siguientes comandos:

- npx eslint .

Este comando no debería mostrar ningún error, puesto que ya todos fueron arreglados
